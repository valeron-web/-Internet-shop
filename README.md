# Интернет магазин

Полнофункциональное веб-приложение для управления дропшиппинг-бизнесом. Проект разработан в качестве портфолио и демонстрирует навыки создания full-stack приложений с использованием современных технологий.

## Функциональные возможности

- **Каталог товаров**: Отображение товаров с фильтрацией по категориям, поиском и сортировкой по цене.
- **Административная панель (CRM)**: Закрытый раздел для управления ассортиментом. Реализован полный цикл CRUD-операций: добавление, редактирование, удаление товаров, а также управление статусом наличия.
- **Система аутентификации**: Защита административной панели с использованием JWT-токенов. Пароли хешируются с помощью bcrypt.
- **Парсер данных**: Отдельный Python-скрипт для парсинга товаров с внешних источников (на примере Mandarake). Скрипт извлекает данные из локальных HTML-файлов, копирует изображения и преобразует их в формат, готовый для импорта в базу данных.
- **Модуль заказов**: На стороне клиента реализована форма сбора контактных данных (телефон, Telegram/WhatsApp) для последующей связи с покупателем.

## Технологический стек

**Бэкенд:**
- Node.js, Express.js
- MySQL, Sequelize ORM
- JWT для аутентификации
- bcrypt для хеширования паролей

**Фронтенд:**
- React.js, React Router
- React Bootstrap
- Axios для HTTP-запросов
- Cookies для хранения токенов

**Парсер (отдельный модуль):**
- Python, BeautifulSoup4
- Pillow для обработки изображений (конвертация AVIF в JPEG)

## Ключевые особенности реализации

- Разделение клиентской и серверной частей.
- Кастомный алгоритм ценообразования для дропшиппинга (наценка + доставка).
- Возможность массового импорта товаров через JSON или прямого добавления через админ-панель.
- Адаптивный интерфейс на основе Bootstrap.

## Структура проекта

- `/server` - серверная часть на Node.js (модели, контроллеры, маршруты, middleware)
- `/client` - клиентская часть на React (компоненты, страницы)
- `/parser` - отдельный Python-скрипт для парсинга и подготовки данных
